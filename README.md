# AlmostaCMS

**Create beautiful websites with built-in editing - completely free, forever.**

AlmostaCMS is a revolutionary website builder that gives you a professional site with a built-in admin panel. No servers, no databases, no monthly fees. Just GitHub Pages and your creativity.

---

## 🌟 Key Features

- **🆓 100% Free** - Host unlimited sites on GitHub Pages at zero cost
- **⚡ Built-in Admin** - Every site includes an admin panel at `/admin`
- **🔒 Secure** - Login with your GitHub account, edit your content
- **🚀 Auto-Deploy** - Changes go live in 30 seconds
- **💾 You Own Everything** - All content lives in your GitHub repository
- **📱 Mobile Friendly** - Responsive admin and templates
- **🎨 Customizable** - Full control over HTML, CSS, and design

## 🚀 Getting Started

### Step 1: Choose a Template

Browse our templates (coming soon):
- **Portfolio** - Showcase your work and skills
- **Landing Page** - Promote your product or service
- **Blog** - Share your thoughts and ideas
- **Link-in-Bio** - All your links in one place

### Step 2: Create Your Site

```bash
# Use GitHub's template feature
# Click "Use this template" on a template repository
# Name your repository (e.g., "my-portfolio")
```

### Step 3: Enable GitHub Pages

1. Go to your new repository Settings → Pages
2. Source: Select "GitHub Actions"
3. Wait 1-2 minutes for deployment

### Step 4: Edit Your Site

Visit your admin panel:
```
https://username.github.io/your-repo-name/admin
```

1. Login with GitHub
2. Edit your content (name, bio, projects, etc.)
3. Click Save
4. Changes go live automatically!

```
You visit /admin
     ↓
Login with GitHub

     ↓

     ↓
GitHub Actions deploys
     ↓
Live in 30 seconds ✨
```

**No coding needed!** Just fill out forms and your site updates automatically.
